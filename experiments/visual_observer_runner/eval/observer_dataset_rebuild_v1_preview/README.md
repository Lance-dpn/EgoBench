# Observer Dataset Rebuild Preview

This directory is an isolated preview area for the new observer-evaluation
dataset rebuild process described in `../eval-process.md`.

It is intentionally separate from the existing directories:

- `observer_problem_set_order1/`
- `observer_problem_set_retail/`
- `observer_problem_set_restaurant/`
- `observer_problem_set_kitchen/`

The files here are for format review only. They are not a finalized GT set and
must not be used as the official observer evaluation dataset until reviewed and
promoted.

Current files:

- `sample_preview.json`: small cross-scenario examples using the new
  `visual_query_v1` plus inline GT structure.
